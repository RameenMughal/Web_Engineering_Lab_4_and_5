# Counter with History

This project is Task 1 of the Final Comprehensive Practice Projects. It focuses on mastering React state management by building a counter that tracks its own history.

Live Demo: https://task1-counter.vercel.app/

## Features
1. Increment / Decrement: Modify the counter value by 1 in real-time.
2. Soft Reset: Returns the counter to 0 while preserving the history log.
3. History Reset: Wipes entire history list.
4. History Tracking: A dynamic list that logs every previous state change.
